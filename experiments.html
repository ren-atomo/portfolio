実験

現在進行中の技術的探求と創作の試み

進行中

リアルタイム音響可視化システム 2.0

TouchDesigner / Python / OSC

hamon.exeの進化版。より複雑な音響解析と視覚表現の関係性を探る。FFT解析の精度向上と、3D空間での音の可視化を実験中。複数の音源を同時に解析し、それぞれの特性に応じた独自の視覚言語を生成する。

目標

音の持つ感情的・物理的特性を、より直感的に理解可能な視覚表現へと変換するシステムの確立

進行中

身体性とデジタルの融合

TouchDesigner / Kinect / MediaPipe

身体の動きをリアルタイムでトラッキングし、その動作パターンから生成される視覚・音響フィードバック。ダンサーの動きを音に変換し、その音をさらに映像として投影する循環的なシステムを構築。

目標

身体表現の新たな拡張可能性を探り、パフォーマンスアートの領域を広げる

計画中

集合的無意識の可視化

TouchDesigner / Web API / Machine Learning

複数人の生体データ（心拍、脳波など）を収集し、その同期性や差異を視覚化。集団での瞑想やセッション時に現れる共鳴現象を捉え、「場」の雰囲気を可視化することを試みる。

目標

非言語的な集団コミュニケーションの存在を証明し、新たな対話の形を提案

進行中

環境音のアーカイブと再構築

Field Recording / Max/MSP / TouchDesigner

都市の様々な場所で収集した環境音を、時間・場所・感情のタグ付けとともにアーカイブ。これらを再構築することで、記憶と場所の関係性を音響的に表現する作品を制作。

目標

場所の持つ「雰囲気」を音として保存し、新たな空間体験を創出する

計画中

テルミン × AI

Arduino / TensorFlow / TouchDesigner

テルミンの非接触インターフェースを活用し、空間での手の動きから音と映像を同時生成。機械学習により、演奏者の癖やパターンを学習し、協調的な音楽生成を行うAIシステムの開発。

目標

人間とAIの創造的な協働関係を、音楽というメディアを通じて探求

進行中

写真の時間軸操作

Python / OpenCV / Processing

「There are me」「Shibuya_No.01」で用いた手法をアルゴリズム化。異なる時間に撮影された写真を、ピクセル単位で織り交ぜることで、時間の重層性を表現する自動化システムを開発。

目標

時間という概念を視覚的に再定義し、写真メディアの新たな表現可能性を開拓

今後の展望

技術は手段であり、目的ではない。

これらの実験を通じて探求しているのは、人と人、人と環境、意識と無意識の間に存在する見えない関係性を可視化・可聴化することである。 TouchDesignerやProcessingといったツールは、その探求のための道具に過ぎない。

最終的には、これらの実験から得られた知見を統合し、 観客が能動的に参加し、共に作品を生成していくようなインタラクティブな体験を創出したい。 それは単なる技術デモンストレーションではなく、人々の意識を変容させ、 新たな認識の扉を開く装置となることを目指している。
